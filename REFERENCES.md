## Dataset

This reproduction uses the RadioML 2018.01A-style HDF5 dataset file:

```text
GOLD_XYZ_OSC.0001_1024.hdf5
```

The dataset is not included in this repository because it is very large. Place it locally at:

```text
data/GOLD_XYZ_OSC.0001_1024.hdf5
```

See the dataset sources associated with the RadioML/deepsig ecosystem and the O'Shea et al. work for access details.
